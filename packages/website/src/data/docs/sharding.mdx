## Sharding

Split your test suite across multiple CI workers:

```bash
# Worker 1 of 4
roadtest --shard=1/4

# Worker 2 of 4
roadtest --shard=2/4
```

Each shard runs its slice of tests independently. Results are written to `.roadtest/results/shard-N-of-M.json`.

### Merging results

After all shards complete, merge results for a combined report:

```bash
roadtest --merge-shards
```

### Example GitHub Actions matrix

```yaml
strategy:
  matrix:
    shard: [1, 2, 3, 4]
steps:
  - run: roadtest --shard=${{ matrix.shard }}/4 --coverage
```
